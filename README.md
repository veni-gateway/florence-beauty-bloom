# Florence Beauty Studio

## About

Florence Beauty Studio is a soft luxury beauty destination offering professional hair, skin, makeup, and bridal services in Rajkot, Gujarat. Founded by Shobhana Parmar, we provide personalized beauty experiences where every client feels valued and celebrated.

## Services

- **Hair Services**: Professional styling, coloring, and treatments
- **Skin Services**: Advanced skincare treatments and therapy  
- **Makeup Services**: Professional makeup application for all occasions
- **Bridal Makeup**: Specialized bridal styling and makeup services
